Boolean Laws (mini)

Identity: A ∧ 1 = A, A ∨ 0 = A
Null: A ∧ 0 = 0, A ∨ 1 = 1
Idempotent: A ∧ A = A, A ∨ A = A
Inverse: A ∧ ¬A = 0, A ∨ ¬A = 1

Bitwise vs Logical

CPU bitwise ops (AND, OR, XOR, NOT) operate on each bit independently. Logical ops evaluate true/false for whole expressions. 

0b1010 AND 0b1100 = 0b1000
0b1010 XOR 0b1100 = 0b0110
Base Conversion (Decimal → Binary)

Repeated division by 2; collect remainders (bottom-up):

13 / 2 = 6 r1
6 / 2  = 3 r0
3 / 2  = 1 r1
1 / 2  = 0 r1
=> 13₁₀ = 1101₂

Study Tips

  1. Memorize powers of two up to at least 210 (1024).
  2. Group binary digits in 3s (octal) or 4s (hex) for quick conversion.
  3. Write out small truth tables to debug logic.

Glossary

  • RAM — volatile working memory.
  • ALU — performs arithmetic/logic ops.
  • MSB / LSB — bit positions.
